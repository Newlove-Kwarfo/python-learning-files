ENGINEERING UNIT CONVERTER

HOW TO RUN THE PROGRAM

- Make sure Python is installed on your computer.
- Open a terminal or command prompt or IDE
- Navigate to the folder where the file is saved.
- Run the program python converter_22306910.py

- Follow the on-screen instructions to choose conversions.
- Select either Temperature or Distance conversion.
- Choose the type of conversion (from which unit to which unit).
- Enter the value to be converted.
- Choose how many decimal places to round the result.
- View the final converted result.


SUPPORTED CONVERSIONS

This program supports the following conversions:

TEMPERATURE:
- Fahrenheit to Celsius
- Celsius to Fahrenheit
DISTANCE:
- Meters to kilometers, miles, feet, and inches
- Kilometers to meters, miles, feet and inches
- Miles to meters, kilometers, feet and inches
- Feet to meters, kilometers, miles and inches
- Inches to meters, kilometers, miles and feet


PROGRAM FEATURES

- Validates all user input
- Rejects invalid menu options
- Prevents negative distance values
- Allows user to choose rounding decimal places
- Provides clear formatted results
- Handles errors such as invalid units
